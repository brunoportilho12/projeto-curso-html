O QUE É HTML:

(Voltar)
HTML significa Hypertext Markup Language. É uma linguagem de marcação usada para criar páginas da web. Ele permite que os desenvolvedores especifiquem os elementos que compõem uma página da Web, como títulos, parágrafos, imagens, links e outros tipos de conteúdo.

HTML é a base do desenvolvimento web, trabalhando com outras linguagens como CSS (Cascading Style Sheets) para definir a aparência e layout de uma página, e JavaScript para adicionar interatividade e dinamismo. As páginas HTML podem ser visualizadas em navegadores da Web, como Google Chrome, Firefox e Internet Explorer.






PRINCIPAIS FERRAMENTAS USADA NO HTML:

(Voltar)
Várias ferramentas estão disponíveis para o desenvolvimento de páginas HTML.

Ambientes de desenvolvimento integrado: IDEs como Visual Studio Code, Atom e Dreamweaver fornecem recursos adicionais para simplificar a escrita de código HTML, como realce de sintaxe, dicas de código e depuração.

Gerenciador de pacotes: gerenciadores de pacotes, como o NPM, ajudam a instalar e gerenciar bibliotecas e pacotes de terceiros usados ​​para desenvolver páginas HTML.

Editores de imagem: Editores de imagem como Photoshop e GIMP podem ser usados ​​para criar e editar imagens a serem usadas em páginas HTML.






INSPETOR DE ELEMENTOS:

(Voltar)

O Inspetor de Elemento é uma ferramenta integrada ao seu navegador da Web que permite inspecionar e modificar o código HTML, CSS e JavaScript de páginas da Web em tempo real.

É útil para desenvolvedores da Web que precisam depurar problemas de layout, inspecionar hierarquias de elementos, visualizar propriedades de estilo e testar alterações de código.

Para acessar o inspetor de elemento, basta clicar com o botão direito do mouse em qualquer lugar da página e selecionar "Inspecionar" no menu que aparece.

Você pode selecionar qualquer elemento na página para inspecionar suas propriedades e alterar o código HTML ou CSS para testar novos estilos ou ajustes de layout.






O QUE SÃO TEGS:

(Voltar)

As tags em HTML são elementos de marcação que indicam ao navegador como interpretar e exibir o conteúdo de uma página web. Cada tag é composta por um nome de tag entre os símbolos «».

Algumas tags também possuem atributos que especificam informações adicionais sobre o elemento.


  • html: define o início e o fim do documento HTML;
  • head: contém informações sobre o documento, como o título da página, meta tags e links para arquivos externos;
  • body: contém todo o conteúdo da página visível ao usuário;
  • h1, h2, h3, h4,h5, h6: define títulos e subtítulos de diferentes tamanhos;
  • p: define um parágrafo de texto;
  • a: define um link para outra página ou recurso;
  • img: define uma imagem;
  • ul e ol: define listas não ordenadas e ordenadas, respectivamente;
  • div e span: são usados para agrupar elementos e aplicar estilos CSS a eles.

Existem muitas outras tags em HTML, cada uma com um propósito específico.

É importante conhecer bem as tags e seus atributos para criar páginas web bem estruturadas e de fácil manutenção.





LISTAS ORDENADAS E NÃO ORDENADAS:

(Voltar)

Em HTML, existem duas tags principais para criar listas: ul e ol.
ul é a tag para criar uma lista não ordenada. Cada item da lista é criado com a tag li.
Por exemplo:
ul
li Item 1 /li
li Item 2 /li
li Item 3 /li
/ul

Nesse exemplo, é criada uma lista com três itens. Cada item é um elemento li, e a lista inteira é envolvida pela tag ul. Quando essa lista é renderizada no navegador, cada item é exibido com um marcador padrão (como um ponto, um quadrado vazio ou um círculo vazio) à esquerda do texto.

Por outro lado, /ol é a tag para criar uma lista ordenada. Cada item da lista é criado com a tag li, da mesma forma que em uma lista não ordenada. A diferença é que os itens são numerados automaticamente pelo navegador. Por exemplo:

ol
li Item 1 /li
li Item 2 /li
li Item 3 /li
/ol

Nesse exemplo, é criada uma lista ordenada com três itens. Cada item é um elemento li, e a lista inteira é envolvida pela tag /ol. Quando essa lista é renderizada no navegador, cada item é exibido com um número à esquerda do texto.

É importante lembrar que as tags /ul e /ol são apenas as tags principais para criar listas em HTML. Há muitas outras tags e atributos disponíveis para personalizar a aparência e o comportamento das listas, como as tags dl, dt e dd para criar listas de definições, e os atributos type e start para personalizar a numeração ou a marcação dos itens.






ATRIBUTOS DAS TAGS:

(Voltar)

As tags em HTML podem ter atributos que fornecem informações adicionais sobre o elemento ou especificam seu comportamento. Os atributos são adicionados à tag como pares nome-valor, separados por um sinal de igual (=). O valor do atributo é sempre colocado entre aspas duplas.
Por exemplo:


a href=" https://www.exemplo.com " Este é um link /a
Nesse exemplo, a tag a tem um atributo href que especifica o URL do link. O valor do atributo é " https://www.exemplo.com ", colocado entre aspas duplas. Alguns dos atributos mais comuns em HTML incluem:
  • class: especifica uma ou mais classes CSS que serão aplicadas ao elemento;

  • id: define um identificador exclusivo para o elemento, que pode ser usado para referenciá-lo em CSS ou JavaScript;

  • rc: especifica o URL de uma imagem, um arquivo de áudio ou vídeo, ou outro recurso externo que deve ser incorporado na página;

  • href: especifica o URL do destino do link;

  • lt: fornece uma descrição alternativa para uma imagem ou outro recurso, que é exibida caso o recurso não possa ser carregado;

  • title: fornece uma dica de ferramenta para o elemento, que é exibida quando o usuário passa o mouse sobre ele.


Há muitos outros atributos disponíveis em HTML, cada um com um propósito específico. É importante usar os atributos de forma consistente e semântica, para garantir que a página seja acessível, bem estruturada e fácil de manter.
